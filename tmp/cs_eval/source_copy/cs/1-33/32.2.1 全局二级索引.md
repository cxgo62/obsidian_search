对于sharding key之外的查询，有两种查询方式：

1.  scatter/gather 对每个分片进行广播，并对结果进行搜集，缺点是性能很差
2.  global secondry index

## term-partitioned 
1.  term-partitioned 一种实现局二级索引的方式，term这个名字来自于全文索引，本质上就是个倒排缩影，second index -> set of primary key
2.  二级索引自身也涉及到分片的问题
3.  二级索引一般是异步写的，否着需要引入分布式事物，如XA[[91 XA 2PC 3PC]]
